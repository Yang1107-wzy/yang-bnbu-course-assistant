# Yang 抢课脚本 v1.2.1 Manual-First Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让已打开的手动详情页先于后台 Worker 扫描和提交，同时移除 Test/立即启动中的校时与 Worker 创建阻塞。

**Architecture:** 未标记的详情页升级为 Hot Page，并按页面类别扫描全部目标；专用 Worker 保持兜底。运行控制先发布，校时和批量 Worker 预热异步进行，所有页面继续共用动作队列、锁、pending action 和动作白名单。

**Tech Stack:** JavaScript ES modules、Tampermonkey GM API、Node.js 内置测试运行器、jsdom、esbuild、ESLint。

## Global Constraints

- `STOPPED` 时 Hot Page 只识别，绝不提交。
- 即时启动不等待 BNBU 校时或 Worker 页面加载。
- 只允许 `selectItem`、`selectItemFromWaiting`、`joinWaiting`。
- 禁止 Drop、Replace、Exit Waiting、隐藏 API、自动登录和验证码处理。
- 保留 ME-1、ME-2、FE-1 三个专用 Worker 与 250 ms 全局提交间隔。
- 所有修改位于 `feat/v1.2.0-worker-pool` 隔离工作树。

- [ ] **Step 1: 写 Hot Page 行为失败测试**

```js
test("an unmarked running detail page acts as the foreground hot page", async () => {
  const dom = await fixture("selectable.html", "https://mis.bnbu.edu.cn/mis/student/es/eleDetail.do?id=me");
  let calls = 0;
  dom.window.selectItem = () => {
    calls += 1;
    return dom.window.confirm("Select Example Major Elective (1001), are you sure?");
  };
  const gm = createGm({
    [CONFIG_KEY_V3]: configWithTargets(DEMO_MAJOR),
    [MIGRATION_KEY_V12]: true,
    [CONTROL_KEY_V3]: {
      version: 3,
      generation: 1,
      mode: "MANUAL",
      running: true,
      scheduleEnabled: false,
      selectionWindows: createDefaultConfig().selectionWindows
    }
  });
  const runtime = await createAssistantRuntime({ pageWindow: dom.window, gm, autoTimers: false, tabId: "manual-tab" });
  await runtime.initialize();
  assert.equal(calls, 1);
});

test("a stopped hot page only identifies and never acts", async () => {
  const dom = await fixture("selectable.html", "https://mis.bnbu.edu.cn/mis/student/es/eleDetail.do?id=me");
  let calls = 0;
  dom.window.selectItem = () => { calls += 1; };
  const gm = createGm({ [CONFIG_KEY_V3]: configWithTargets(DEMO_MAJOR), [MIGRATION_KEY_V12]: true });
  const runtime = await createAssistantRuntime({ pageWindow: dom.window, gm, autoTimers: false, tabId: "manual-tab" });
  await runtime.initialize();
  assert.equal(calls, 0);
  assert.match(dom.window.document.querySelector("#yang-worker-status").textContent, /前台优先页/);
});
```

增加一个包含两门 ME 目标的 fixture/配置，断言一次扫描更新两门状态。
