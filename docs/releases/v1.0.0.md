# Yang 抢课脚本 v1.0.0

首个公开稳定版本，面向 BNBU MIS 的 Chrome + Tampermonkey 可视化选课与轮候流程。

## Highlights

- 支持`立即启动`和三个可编辑北京时间窗口的`预约启动`。
- 自动识别 Select、Select from Waiting 和 Join Waiting List。
- 多目标独立扫描、ME/FE worker 协调和全局 FIFO 串行提交。
- 公开版只含虚构示例课程，不保存登录信息或个人资料。
- 附带完整源码 ZIP、独立 userscript 和 SHA-256 校验文件。

## Install

安装 `yang-bnbu-course-assistant.user.js` 后，先进入`设置`删除示例并填写自己的课程，再运行 `Test`。只启用一个版本。

## Verification

发布资产通过 ESLint、Node 自动化测试、esbuild 构建、dist 安全审计、ZIP 内容检查和 SHA-256 回验。

## Known limitation

本版本发布时，2026 年 7 月 20 日正式选课窗口尚未开放，因此尚未完成真实账号的最终提交验收。页面结构或确认文字变化时，脚本默认停止而不是放宽匹配。

使用者必须自行遵守学校规则；软件不保证获得课程名额。
