# Yang 抢课脚本 v1.1.0

本版本改善浏览器内的长期使用体验，并预填三门目标课程。

## Highlights

- 标题栏拖动、右下角自由缩放及自动视口边界保护。
- 收起为可拖动的 `Yang` 悬浮按钮，展开后恢复原尺寸。
- 跨 MIS 页面、刷新和浏览器重启保存布局。
- Tampermonkey 菜单可强制展开或重置面板。
- 默认预填 AI3133 (1001)、COMP4213 (1001) 和 EBIS3113 (1002)。
- 只迁移未经修改的旧 DEMO 目标，不覆盖用户自定义课程。

## Install

安装 `yang-bnbu-course-assistant.user.js` 后刷新所有 MIS 标签，确认只启用一个版本。先运行 `Test`，核对三门目标唯一匹配且页面函数显示 `READY`，再选择立即启动或预约启动。

## Verification

发布资产通过 ESLint、Node 自动化测试、esbuild 构建、dist 安全审计、ZIP 内容检查和 SHA-256 回验。
