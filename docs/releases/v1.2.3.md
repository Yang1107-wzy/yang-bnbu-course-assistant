# Yang 抢课脚本 v1.2.3

> 仅供学习交流，禁止商业使用，不得用于学校正式选课或其他真实教务操作。请遵守中国法律法规、学校规章及系统使用政策。

## 本版修复

- Worker 拓扑改为每类别一个：`ME-1` 遍历所有 ME 目标，`FE-1` 遍历所有 FE 目标。
- 手动打开的 ME/FE 详情页会登记为健康 Hot Page，阻止同类别重复开页。
- Worker 15 秒未认领时显示明确失败，按 5/15/30/60 秒退避重试。
- Worker URL 不再携带课程列表，身份改为类别与 generation，降低重定向和配置变更导致的认领失败。
- 保留前台页优先、250 ms FIFO、Join Waiting List 及 Select from Waiting。

## 升级注意

升级会清除 v1.2.2 的旧 Worker 注册和待执行队列，但保留目标课程、时间窗口、面板布局和已接受的使用声明。首次加载 v1.2.3 会处于 `STOPPED`。
