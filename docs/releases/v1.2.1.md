# Yang 抢课脚本 v1.2.1

v1.2.1 将用户手动打开的 ME/FE 详情页设为前台优先页：当前页面已经显示目标课程时，脚本先遍历本类别全部目标，再异步补建缺失的专用 Worker。

## 主要变化

- `Test` 立即返回当前页面识别结果，不再等待三个后台标签加载。
- `立即启动`先发布 RUNNING 并扫描当前页；服务器校时和 Worker 预热不阻塞 Select 或 Join Waiting List。
- BNBU 同源校时增加 1 秒硬超时，失败时明确回退本机时间。
- 三个 Worker Opening Lease 使用一次批量存储写入，连续启动不会逐个往返创建。
- 前台页候选优先于尚未提交的相同后台候选，继续共用 250 ms 动作间隔、动作白名单和结果验证。
- `STOPPED` 时前台优先页只识别，不会自动提交。

## 安装与升级

安装 `yang-bnbu-course-assistant.user.js` 后刷新所有 BNBU MIS 标签，并确认 Tampermonkey 只启用一个版本。可先手动打开 ME/FE 详情页，确认迷你条显示“Yang 前台优先页”，再点击 Test 或启动。

本版本仍未在 2026 年 7 月 20 日正式选课窗口完成真实提交验收，不保证获得课程名额。
